# SQL Practice Repository

## 📌 About This Repository

Welcome to my SQL Practice Repository.

This repository documents my SQL learning journey from basic queries to intermediate business analysis using PostgreSQL.

The purpose of this repository is to build strong SQL fundamentals through consistent daily practice and real-world business scenarios.

Every folder contains:

- SQL source code
- Practice questions
- Query outputs
- Screenshots
- README documentation
