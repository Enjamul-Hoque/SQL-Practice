# Online Bookstore SQL Project

## Project Overview

This project demonstrates SQL skills using an Online Bookstore database in PostgreSQL.

## Skills Used

- CREATE TABLE
- INSERT
- COPY
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- INNER JOIN
- Aggregate Functions

## Business Problems Solved

- Find total revenue
- Analyze customer orders
- Find top-selling books
- Analyze books by genre
- Calculate average book price

## Tools

- PostgreSQL
- pgAdmin 4
- GitHub
